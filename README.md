# albinotonnina.com

## Overview

Src files for [www.albinotonnina.com](http://www.albinotonnina.com)

## Install
	$ npm install

## Dev
    $ npm run dev

## Build
	$ npm run build

## Test
	$ npm test
 ([standard](https://standardjs.com) && jest)

### Test Coverage
    $ npm run coverage

### Bundle analysis
	$ npm run stats
